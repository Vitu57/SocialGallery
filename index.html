SocialGallery

Introducción

Mi proyecto llamado Social Gallery consiste en un proyecto que involucra la conexión de una página web php a una base de datos, en la que posteriórmente le añadiremos un login con una tabla usuarios donde cada usuario podrá colgar en su sesión las imagenes que el usuario quiera, y que estas imágenes una vez en la base de datos se muestren en la página de Social Gallery.

Explicación del proyecto

Para poder entrar en la página web hace falta Registrarse o tener cuenta en la base de datos, arriba a la derecha he puesto el href para registrase, en el caso de tener cuenta entrarías dentro la pagina. En el caso de faltar campos necesarios o introducir datos incorrectos la validación con Javascript será incorrecta y te saldrá un error como el foto 2.1
